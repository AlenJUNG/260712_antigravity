# 🚀 CLI 환경 즉시 배포 및 링크 생성 가이드

로컬 터미널이나 CLI 환경(예: `agy cli`)에서 웹페이지(HTML/CSS/JS)를 즉시 배포하고 다른 사람에게 공유할 수 있는 링크를 생성하는 가이드입니다. 

기존에 설정된 **Netlify**를 활용하는 방법과 가장 간편한 **Surge**를 이용하는 방법을 소개합니다.

---

## 🗺️ 배포 방식 요약 비교

| 항목 | 🌐 Netlify CLI | ⚡ Surge CLI |
| :--- | :--- | :--- |
| **특징** | 기설정된 사이트 ID 연동, 안정적인 지속 배포 | 회원가입/설정 없이 터미널에서 즉시 배포 |
| **속도** | 보통 (브라우저 로그인 필요) | 매우 빠름 (이메일/비밀번호만 입력) |
| **로그인 방식** | 브라우저 연동 인증 (`netlify login`) | 터미널 대화형 입력 (`surge login`) |
| **사용 환경** | 정식 프로젝트 배포, 커스텀 도메인 연동 | 빠른 테스트용 임시 주소 발급 |

---

## 1. 🌐 Netlify CLI를 사용한 배포
현재 프로젝트에는 이미 Netlify 설정 파일(`.netlify/state.json`)이 있어, 기존 사이트 ID(`42529771-b4dd-4db2-bb41-bfc501a6a4a6`)로 손쉽게 배포할 수 있습니다.

### Step 1. Netlify CLI 로그인
터미널에서 아래 명령을 실행합니다. 자동으로 브라우저 창이 열리며 Netlify 인증을 진행합니다.
```powershell
npx netlify login
```

### Step 2. 드래프트(테스트) 배포
배포하고자 하는 디렉토리(여기서는 `docs` 폴더)를 지정하여 배포합니다.
```powershell
npx netlify deploy --dir=docs
```
* 명령어가 성공적으로 끝나면 터미널에 **Website Draft URL**이 표시됩니다. 이를 통해 변경 사항을 사전에 확인합니다.

### Step 3. 프로덕션(최종) 배포
최종 퍼블리싱 상태로 올리고 고정된 공식 배포 주소를 발급받으려면 `--prod` 플래그를 붙여 실행합니다.
```powershell
npx netlify deploy --dir=docs --prod
```

## 2. ⚡ Surge CLI를 사용한 즉시 배포
복잡한 계정 로그인이나 사전 연동 과정 없이, **단 3초 만에 임시 공유 링크**를 생성하고 싶을 때 강력히 추천하는 도구입니다.

### Step 1. Surge 배포 실행
터미널에 아래 명령어를 실행하고 배포할 디렉토리를 지정합니다.
```powershell
npx surge ./docs
```

### Step 2. 터미널 내 즉시 계정 생성 (최초 1회)
명령어를 실행하면 터미널에서 즉시 이메일과 비밀번호 입력을 요구합니다.
```text
   email: user@example.com
   password: **********
```
* 최초 입력한 이메일과 비밀번호로 Surge 계정이 자동 생성되며 바로 배포가 시작됩니다.
* 두 번째 실행부터는 기존 계정 정보가 유지되어 추가 입력 없이 즉시 배포됩니다.

### Step 3. 도메인 지정 및 공유
계정 인증이 끝나면 배포 경로와 함께 임의로 생성된 서브도메인이 표시됩니다.
```text
   project: ./docs
   domain: funny-pear.surge.sh
```
* 원하는 도메인 주소(예: `my-test-page.surge.sh`)가 있다면 텍스트를 직접 입력해 바꿀 수도 있습니다.
* 엔터를 누르면 배포가 완료되며, 출력된 링크(`https://funny-pear.surge.sh`)를 복사해 다른 사람에게 전달하면 즉시 웹사이트를 확인할 수 있습니다.

## 💡 에이전트 자동화 꿀팁
`agy cli` 내에서 에이전트(AI)에게 명령하여 대신 배포를 처리하도록 할 수도 있습니다. 
예를 들어, 에이전트에게 아래와 같이 자유롭게 명령해 보세요:

> **"docs 폴더를 surge로 배포하고 링크를 공유해 줘."**  
> (에이전트가 `npx surge ./docs` 커맨드를 생성하고 배포가 성공하면 생성된 링크를 텍스트로 깔끔하게 정리해 보여줍니다.)
