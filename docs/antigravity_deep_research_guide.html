🔍 Google Antigravity CLI (agy) Deep Research (딥 리서치) 활용 가이드

Google Antigravity CLI(agy)의 Deep Research (딥 리서치) 기능은 백그라운드에서 실행되는 독립적인 서브에이전트(Subagent)를 가동하여, 다각도로 웹 검색을 수행하고 신뢰할 수 있는 소스를 교차 검증한 후 심층 리포트를 자동으로 작성하는 강력한 기능입니다.

이 가이드는 agy에서 딥 리서치 기능을 실행하고 관리하는 구체적인 방법을 다룹니다.

1. 💬 대화형 콘솔 (TUI)에서의 실행 방법

agy를 단독으로 실행한 대화형 터미널 사용자 인터페이스(TUI) 모드에서는 슬래시 명령어를 사용해 리서치를 즉시 시작할 수 있습니다.

1) 기본 명령어 형식

/research [주제 또는 질문]
  • 예시:
    bash /research 2026년 최신 Model Context Protocol (MCP) 명세와 보안 가이드라인

2) 동작 흐름

  1. /research 명령어를 입력하면 agy 내부에서 agy:runner 또는 리서치 전문 서브에이전트를 백그라운드에 생성합니다.
  2. 해당 서브에이전트가 백그라운드에서 목표 사이트들을 탐색, 스크랩 및 분석을 비동기(Asynchronous)로 수행합니다.
  3. 리서치가 완료되면 배경 정보, 핵심 조사 결과, 한계점, 검증된 출처(Citations)가 포함된 체계적인 리포트를 받아볼 수 있습니다.
  4. 리서치가 진행되는 동안 개발자는 차단(Block)되지 않고 메인 터미널에서 계속 코딩이나 대화를 진행할 수 있습니다.

2. 💻 터미널에서의 단발성 실행 방법 (Non-interactive Mode)

대화형 콘솔에 들어가지 않고 터미널에서 한 번에 리서치 명령을 보낸 뒤 결과를 텍스트로 바로 출력받을 수 있습니다.

agy -p "/research [주제]"
# 또는
agy --print "/research [주제]"
  • 예시:
    bash agy -p "/research React 19의 변경 사항과 아키텍처적 차이점"

3. 🔌 Claude Code 등 타 에디터 연동 환경에서의 실행 방법

antigravity-plugin-cc 등의 플러그인을 사용하여 Claude Code 터미널 환경에서 agy 기능을 연동한 경우, 접두사를 붙여 딥 리서치 기능을 수행할 수 있습니다.

/agy:research [주제]
  • 예시:
    bash /agy:research WebGPU API의 최신 브라우저 지원 현황 및 WebGL 대비 성능 메트릭
  • 장점: 토큰 소모량이 큰 심층 리포트 작성 작업을 agy(Gemini 백엔드)에 위임(Delegate)하여 자원을 절약하고 보다 풍부한 검색 결과를 획득할 수 있습니다.

4. ⚙️ 백그라운드 리서치 모니터링 및 관리

딥 리서치는 백그라운드에서 오랜 시간(수십 초 ~ 수 분) 동작할 수 있으므로 관리 명령어를 알아두면 좋습니다.

  • /agents (또는 /tasks): 백그라운드에서 활성화되어 작동 중인 서브에이전트 및 리서치 작업 목록을 조회합니다.
  • 에이전트 상세보기 (Deep Monitoring): /agents 패널에서 동작 중인 특정 서브에이전트를 선택하면 해당 에이전트가 무슨 생각을 하고 있으며 어떤 검색을 수행 중인지 로그와 실행 단계를 실시간으로 모니터링할 수 있습니다.
  • /delegate [작업]: 리서치 외에 범용적인 백그라운드 작업을 에이전트에게 지시하여 리소스를 유연하게 활용할 수 있습니다.
