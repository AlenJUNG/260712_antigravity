# 🔍 Google Antigravity CLI (agy) Deep Research (딥 리서치) 활용 가이드

Google Antigravity CLI(`agy`)에서 **Deep Research (딥 리서치)** 기능은 별도의 물리적인 셸 서브커맨드나 TUI 슬래시 명령어(예: `/research`)로 존재하는 것이 아니라, **에이전트 고유의 백그라운드 태스크 및 서브에이전트 가동 능력**을 통해 자연스럽게 작동하는 기능입니다.

이 가이드는 `agy` 환경에서 딥 리서치를 가장 효과적으로 실행하고 활용하는 방법을 설명합니다.

---

## 1. 💬 자연어 프롬프트를 통한 실행 (TUI 및 단발 실행 공통)

`agy` 에이전트는 내부적으로 `research`라는 강력한 읽기/웹 검색 전용 서브에이전트를 내장하고 있습니다. 따라서 별도의 명령어 형식 대신, **자연어 프롬프트로 리서치를 명시적으로 지시**하면 자동으로 딥 리서치 모드가 가동됩니다.

### 1) TUI 대화창에서 사용 시
`agy` 콘솔을 연 후 아래와 같이 입력합니다.
*   **프롬프트 예시:**
    ```text
    "2026년 최신 Model Context Protocol (MCP) 표준 규격에 대해 딥 리서치(Deep Research)를 수행하고 결과를 정리해줘."
    ```
    ```text
    "웹 검색과 공식 문서를 통해 React 19 아키텍처의 변경점을 심층 조사해서 보고서 형태로 작성해줘."
    ```

### 2) 터미널 단발성 실행 시 (Non-interactive Mode)
TUI에 진입하지 않고 셸에서 결과를 바로 보려면 `-p` (또는 `--print`) 플래그를 사용합니다.
*   **커맨드 예시:**
    ```bash
    agy -p "WebGPU API의 최신 브라우저 지원 현황에 대해 딥 리서치해서 요약해줘"
    ```

---

## 2. 🎯 장시간 정밀 조사를 위한 `/goal` 명령어 활용

단순한 질문-답변 수준을 넘어, 에이전트가 끊임없이 리서치 경로를 검증하고 상세한 보고서 파일을 생성할 때까지 동작하게 하려면 중급자용 명령어인 `/goal`을 병용하는 것이 베스트입니다.

1.  `agy` 대화창에서 `/goal`을 입력합니다.
2.  에이전트가 목표를 물어보면 다음과 같이 구체적으로 리서치 목표를 지정합니다:
    *   **목표 지정 예시:**
        ```text
        목표: 'AI 영상 편집 파이프라인 최적화 도구'에 대해 해외 테크 블로그 및 공식 문서를 딥 리서치하고, 분석 결과를 ./docs/ai_video_research_report.md 파일로 완벽하게 작성해줘.
        ```
3.  에이전트는 목표가 달성될 때까지 백그라운드에서 브라우징 및 교차 검증을 반복 실행합니다.

---

## 3. ⚙️ 내부 동작 원리 및 모니터링

*   **서브에이전트 위임:** 사용자의 리서치 프롬프트를 받으면 메인 에이전트는 독립적인 `research` 컨텍스트를 분리하여 가동합니다.
*   **작업 모니터링 (`/agents` 또는 `/tasks`)**: 리서치 서브에이전트가 동작하는 동안 대화창이 멈추지 않으며, 언제든지 `/agents` 명령어를 쳐서 백그라운드 리서치 진행 로그와 생각을 모니터링할 수 있습니다.
