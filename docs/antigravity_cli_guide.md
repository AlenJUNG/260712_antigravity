# 💻 Google Antigravity CLI (agy) 레벨별 활용 가이드

Google Antigravity CLI(`agy`)의 실행 플래그 및 콘솔 내 명령어를 사용자 숙련도(초급/중급/고급)에 맞춰 체계적으로 정리한 실전 가이드입니다.

---

## 1. 🟢 초급자 레벨 (Beginner): 기본 실행 및 상태 관리

터미널에서 에이전트와 대화를 시작하고 기본적인 세션 및 상태를 관리하는 단계입니다.

### 1. agy 기본 실행 및 세션 복구
*   **`agy`**: 기본 대화 세션을 새로 시작합니다.
*   **`agy --continue` (단축형: `agy -c`)** 🌟 **[초급 꿀팁]**
    *   **설명**: 매번 세션을 새로 파지 않고, **가장 최근에 나누었던 대화 맥락을 그대로 이어서 진행**합니다.
    *   **팁**: 오류를 고치는 도중 창이 닫혔거나, 어제 하던 작업을 연이어 코딩할 때 무조건 이 커맨드를 사용해 컨텍스트 손실을 막으세요.

### 2. 단발성 질의 (프린트 모드)
*   **`agy --print "질문"` (단축형: `agy -p "질문"`)**
    *   **설명**: 대화 인터랙티브 콘솔을 열지 않고, 터미널에서 질문 하나만 바로 던진 뒤 답변을 텍스트로 출력받고 끝냅니다.
    *   **팁**: 단순 스크립트 작성이나 특정 패키지 설치 여부 등을 셸 커맨드로 한 번에 물어볼 때 매우 유용합니다.

### 3. 상태 및 사용량 조회 (슬래시 명령어)
*   **`/usage`**: 주간 및 5시간 단위로 충전되는 AI 모델 쿼타(Quota) 남은 사용량을 그래픽 게이지로 확인합니다.
*   **`/context`**: 현재 에이전트의 대화방 메모리 크기(토큰 수)와 최대 용량 대비 점유율을 확인합니다. 방이 너무 꽉 차서 기억력이 감퇴할 때 확인해 봅니다.
*   **`/clear`**: 콘솔 화면을 깨끗하게 비웁니다.
*   **`/exit` (또는 `/quit`)**: 에이전트 세션을 종료하고 본래 셸 터미널로 돌아갑니다.

## 2. 🟡 중급자 레벨 (Intermediate): 고도화된 기능 및 설계 조율

코드를 직접 수정하기 전 설계 단계를 밟거나 에이전트에게 주도권을 주어 최적의 답안을 도출해 내는 고급 조율 기능입니다.

### 1. 안전한 설계 선행 (`/plan`) 🌟 **[중급 필수]**
*   **설명**: 에이전트에게 본 코드를 즉각 수정하게 하지 않고, **'구현 계획서(Plan)'부터 작성하도록 강제**합니다.
*   **꿀팁**: 복잡한 비즈니스 로직이나 의존성 수정 작업을 요청하기 전에 `/plan`을 먼저 실행시키면 에이전트가 구조적 결함을 미리 발견하고 실수를 대폭 줄여줍니다.

### 2. 인터뷰 모드 (`/grill-me`)
*   **설명**: 에이전트가 개발자에게 역으로 질문을 던져 설계 의도나 프로젝트 스펙을 꼼꼼하게 인터뷰해 갑니다.
*   **꿀팁**: 기획서나 마크다운 명세서가 조금 모호할 때 `/grill-me`를 시작하면, 에이전트가 놓쳤던 세부 사안(예: "에러 시 디폴트 리턴값은 뭘로 할까요?", "어떤 라이브러리를 선호하나요?")을 조율한 뒤 최종 코딩을 시작합니다.

### 3. 목표 완수 모드 (`/goal`)
*   **설명**: 에이전트에게 1회성 명령어가 아닌, **특정 목표가 최종 성취될 때까지 스스로 도구를 실행하고 검증을 반복하는 모드**를 부여합니다.
*   **꿀팁**: 밤새 테스트 커버리지를 90%로 올리기, 레거시 코드 일괄 리팩토링하기 등 시간이 오래 걸리고 지속적인 시도가 필요한 '장기 목표'를 던져둘 때 사용합니다.

### 4. 모델 변경 (`/model` 및 `/models`)
*   **설명**: 현재 열려있는 세션 중에 사용 모델을 즉석에서 변경하거나 사용 가능한 모델 목록을 조회합니다.
*   **꿀팁**: 평소에는 저렴하고 빠른 `Gemini 3.5 Flash (Medium)`을 쓰다가 복잡한 알고리즘이나 수학 연산, 대형 리팩토링 단계에서는 `/model Gemini 3.5 Pro (High)`로 전술을 변경할 수 있습니다.

## 3. 🔴 고급자 레벨 (Advanced): 완전 자동화, 보안 및 지식 확장

API 연동, 플러그인 제어, 대화 영구 내재화 등 생산성을 극한으로 올리는 기법입니다.

### 1. 자동 권한 건너뛰기 (`--dangerously-skip-permissions`) 🌟 **[고급자 꿀팁]**
*   **설명**: 에이전트가 파일 쓰기, 명령어 실행 등 보안상 확인이 필요한 도구를 호출할 때 **매번 개발자에게 묻는 '승인(Approve) 팝업'을 완전히 건너뛰고 자동 실행**합니다.
*   **주의**: 에이전트가 셸 스크립트를 작성하여 바로 실행할 수도 있으므로, 안전하고 믿을 수 있는 샌드박스 상태나 전적으로 검증된 프롬프트 자동화 파이프라인에서만 신중하게 사용해야 합니다.

### 2. 에이전트 학습 및 룰 내재화 (`/learn`)
*   **설명**: 이번 세션에서 에이전트와 치열하게 맞춰놓은 커스텀 규칙이나 수정 법안을 에이전트의 **영구 저장 기억 공간(Knowledge Base)**에 등록시킵니다.
*   **꿀팁**: "앞으로 HTML 문서는 무조건 `./docs` 하위에 저장하라" 등 한 번 교육해 놓은 지침이 다음 채팅방을 새로 열어도 (`agy` 실행 시) 항상 영구히 적용되게 만드는 마법의 키워드입니다.

### 3. Model Context Protocol 연동 (`/plugins` 및 `agy plugin`)
*   **설명**: 에이전트에게 전용 데이터베이스 조회 툴, Jira 티켓 확인 툴, 특정 API 호출 권한 등 외부 플러그인을 주입해 에이전트의 손발을 확장합니다.

### 4. 특정 과거 대화 복원 (`agy --conversation [ID]`)
*   **설명**: 과거의 특정 대화방 ID를 인자로 넘겨, 그 시점으로 에이전트의 생각과 대화 맥락을 복원시켜 작업을 재개합니다.

### 5. 샌드박스 격리 모드 (`agy --sandbox`)
*   **설명**: 터미널 커맨드 등 시스템 영향력을 제한한 엄격한 격리 환경에서 에이전트를 가동합니다. 외부 오픈소스 코드를 가져와 안전하게 로직만 분석시킬 때 활용합니다.
