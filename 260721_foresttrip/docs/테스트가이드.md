## 📡 Step 3. REST API 기능 테스트 (새 터미널 창에서 진행)

**새 PowerShell 창**을 열어 아래 명령어를 하나씩 복사해서 붙여넣으며 테스트합니다.

---

### ① 서버 헬스 체크 (공개 엔드포인트)
인증 토큰 없이 서버 생존 상태를 확인합니다.

```powershell
Invoke-RestMethod -Uri http://localhost:3000/health
```

- **기대 결과**:
  ```json
  { "ok": true }
  ```

---

### ② v1 상세 헬스 체크 (인증 필수)
Bearer 토큰을 동봉하여 DB 세션 및 모니터링 상태를 확인합니다.

```powershell
$headers = @{ Authorization = "Bearer test-token-very-long-and-secure-2026-auth" }
Invoke-RestMethod -Uri http://localhost:3000/v1/health -Headers $headers
```

- **기대 결과**:
  ```json
  {
    "ok": true,
    "session": { "loggedIn": false, "valid": false, "needsRelogin": false },
    "coverage": { ... }
  }
  ```

---

### ③ 시/도 목록 조회
```powershell
$headers = @{ Authorization = "Bearer test-token-very-long-and-secure-2026-auth" }
Invoke-RestMethod -Uri http://localhost:3000/v1/sido -Headers $headers
```

- **기대 결과**: 전국 9개 시/도 코드(서울/경기, 강원, 충북 등) 목록 반환

---

### ④ 시/도별 휴양림 목록 조회 (예: 강원도 = sido 2)
```powershell
$headers = @{ Authorization = "Bearer test-token-very-long-and-secure-2026-auth" }
Invoke-RestMethod -Uri http://localhost:3000/v1/forests?sido=2 -Headers $headers
```

- **기대 결과**: 강원 지역 자연휴양림 목록, 타입(국립/공립/사립), 개관연도 및 신축 여부 리스트 반환

---

### ⑤ 특정 휴양림 달력 정보 조회 (예: 대관령자연휴양림 = 0111)
```powershell
$headers = @{ Authorization = "Bearer test-token-very-long-and-secure-2026-auth" }
Invoke-RestMethod -Uri http://localhost:3000/v1/forests/0111/calendar -Headers $headers
```

- **기대 결과**: 대관령휴양림의 날짜별 구별(선착순/추첨/휴무) 정보 리스트 반환

---

### ⑥ 전국/지역 실시간 예약 가능 스냅샷 조회
```powershell
$headers = @{ Authorization = "Bearer test-token-very-long-and-secure-2026-auth" }
Invoke-RestMethod -Uri "http://localhost:3000/v1/search?date=20260819&nights=1&nofpr=2&sido=2" -Headers $headers
```

- **기대 결과**: 해당 날짜의 강원 지역 휴양림별 예약가능 여부 요약 및 신선도 타임스탬프 반환

---

### ⑦ 특정 휴양림 실시간 방 목록 상세 조회 (온디맨드)
*숲나들e 사이트에서 실제 로그인 세션을 이용해 대관령(0111) 휴양림의 모든 방 목록을 긁어옵니다.*

```powershell
$headers = @{ Authorization = "Bearer test-token-very-long-and-secure-2026-auth" }
Invoke-RestMethod -Uri "http://localhost:3000/v1/forests/0111/rooms?date=20260819&nights=1&nofpr=2" -Headers $headers
```

- **기대 결과**: 대관령휴양림 각 방의 이름, 정원, 가격, 예약가능 여부 및 대기 순번 리스트 반환 (`pagesTraversed` 페이징 완료 포함)

---

### ⑧ 알림 감시(Watch) 등록 테스트
특정 휴양림 방에 대해 취소표/대기표 모니터링을 등록합니다.

```powershell
$headers = @{ Authorization = "Bearer test-token-very-long-and-secure-2026-auth" }
$body = @{
    deviceId = "test-laptop-device-01"
    type = "forest"
    insttId = "0111"
    date = "20260819"
    nights = 1
    nofpr = 2
    notifyGrades = @("예약가능", "대기가능")
} | ConvertTo-Json

Invoke-RestMethod -Uri http://localhost:3000/v1/watches -Method POST -Headers $headers -Body $body -ContentType "application/json; charset=utf-8"
```

- **등록 후 동작 확인**:
  - 감시가 등록되면 스케줄러가 백그라운드에서 해당 날짜를 주기적으로 모니터링합니다.
  - 상태 변화(예약가능/대기가능) 감지 시 `02_구현\data\pushes.log`에 푸시 로그가 기록됩니다.
